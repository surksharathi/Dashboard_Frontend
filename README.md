# React Frontend Application for Dashboard

This is a basic React application that uses Ant Design and Redux Saga for management of the state.

### Tools Used

- React
- Redux
- Redux Saga
- React Router v4
- Ant Design
- Rechart
- Jest
- Enzyme

### Installing

clone the repo

```
git clone https://github.com/surksharathi/Dashboard_Frontend.git
```

install node modules

```
npm install
```

## Folder Structure

After creation, your project should look like this:

```
Dashboard_frontend/
    node_modules/
    public/
    src/
    	assets/
    	routes/
        store/
        	store1/
            	actions.js
            	reducers.js
        ...
        index.js
        setupTests.js
    .gitignore
    package.json
    README.md
```

## Run the project

```
npm start
```

## Building the project

```
npm run build
```

## For Unit Testing

```
npm test
```
